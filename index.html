Intro to Programming:

Lesson 01 Concepts

1) Thinking like a Programmer
If you want to "think like a programmer", practice the following ways of thinking; procedural thinking, abstract thinking, systematic debugging, systematic thinking, and technological empathy.


2) Tags
Tags are used to write HTML. They can be used to bold or italicize text.


3) Attributes
Attributes can be used to add links or images.


4) New Terms
HTML - HyperText Markup Language
HTTP - HyperText Transfer Protocol. The main protocol of the web.
Servers - computers that host the files that make up the web.
Internet - the world's largest computer network.
Browser - a program that runs on your computer to display files found on the web.
